# Mbokane-MI-FUTURE_DS_01
Future Interns Task 1 | Business Sales Performance Analytics | Power BI

**Project Overview**
This project focuses on analyzing business sales data using Power BI to uncover meaningful insights that support data-driven decision-making. The dashboard provides a comprehensive view of revenue performance, product trends, and regional sales distribution.

**Business Questions Addressed**
The project aims to answer the following key business questions:
Which products generate the most revenue?
What are the top-performing categories?
How does revenue trend over time?
Which regions contribute the most to sales?
What are the patterns in customer purchasing behavior?

**Data Preparation**
The dataset was cleaned and prepared before analysis:
Removed duplicates and missing values
Standardized column names and formats
Converted data types (e.g., dates, currency)
Created calculated columns and measures in Power BI
Structured data into a usable model for analysis

**Dashboard Features**
The Power BI dashboard includes:
Revenue trends over time (line charts)
Top-selling products and categories (bar charts)
Regional performance analysis (maps or charts)
KPI indicators (total revenue, sales count, etc.)
Interactive filters (date, region, product category)

**Key Insights**
Some important insights derived from the analysis:
A small number of products contribute to a large portion of revenue
Certain categories consistently outperform others
Sales show seasonal trends across specific months
Some regions significantly outperform others in revenue generation

**Findings**
High-value products drive overall profitability

**Conclusion**
This project highlights how data analysis can be used to extract actionable insights from business sales data. The Power BI dashboard enables stakeholders to make informed decisions, optimize strategies, and improve overall business performance.
